Identificador de Par ou Ímpar em Java
Um pequeno programa interativo para verificar a paridade de números inteiros.

Funcionalidades:

Recebe um número do usuário.

Informa se o número é PAR ou ÍMPAR.

Menu de repetição para testar vários números em sequência.

Validação simples de opção de saída.

Lógica Utilizada:

Operador % (Mod): Utilizado para verificar o resto da divisão por 2. Se o resto for 0, o número é par; caso contrário, é ímpar.

Estrutura condicional if/else: Para exibir a mensagem correta ao usuário.

Laço do-while: Mantém o programa rodando conforme a vontade do usuário.
